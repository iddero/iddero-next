Compact web server for smartphone or tablet operation

A complete solution for control and monitoring of KNX installations, with all the functionality you need. Intuitive, user-friendly visualisation from any smartphone or tablet (iOS, Android), or from any device equipped with a supported web browser. Remote access through the idderocloud service.

Direct connection to the KNX bus. Includes 8 multi-function inputs: 4 binary inputs, plus 4 configurable binary or temperature probe inputs.

Easy, intuitive configuration

Create your projects in a matter of minutes using the iddero-config configuration software, even without actual access to the target device. Ultra-fast programming over IP.

Accessories

  • DW-TS-N1P

    Temperature probes

  • DW-GSM

    GSM expansion module
