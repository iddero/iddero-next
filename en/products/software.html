iddero config

Software de configuración Iddero

Create your projects in minutes:

  • Import group addresses from ETS
  • Define floorplans, zones, and devices to be controlled
  • Configure each element or function in a user-friendly, intuitive way
  • Associate communication objects and group addresses using a simple drag & drop interface
  • Transfer configuration data to the target device(*) easily over TCP/IP
  • The visualisation is automatically generated from project data; no design work is required.
(*) Available for HCx-KNX touch panels and for Iddero Home Server
